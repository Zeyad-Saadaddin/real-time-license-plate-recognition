# Real-Time License Plate Recognition

Computer vision project focused on real-time license plate detection and text recognition using YOLOv5 and easyOCR.

## Project Overview
This project presents a real-time license plate recognition system developed for automated detection and text extraction from vehicle license plates. The workflow combines object detection and optical character recognition in one computer vision pipeline.

## Key Objectives
- Detect vehicle license plates in real time
- Extract and recognize plate text automatically
- Build a structured workflow for dataset preparation, training, validation, and testing
- Evaluate system performance under practical conditions

## Core Components
- YOLOv5-based license plate detection
- easyOCR-based text recognition
- OpenCV-based image handling and preprocessing
- Dataset collection, labeling, training, validation, and testing

## Main Focus Areas
- Computer vision
- Object detection
- Optical character recognition
- Model evaluation
- Real-time system workflow
